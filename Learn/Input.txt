 EmacsやAtomでの記号の入力の仕方
 https://agda.readthedocs.io/en/latest/tools/emacs-mode.html

 リストはここを参照
 https://github.com/agda/agda/blob/master/src/data/emacs-mode/agda-input.el

 "\b" + 英数字 => 英数字の黒板太字 (ℕ, ℂ, 𝕚, 𝟎)
 "\bN" => "ℕ"
 "\b0" => "𝟘"

 矢印
 関数型で使われる矢印は "→"
 "\to" もしくは "\r" => "→"
 "\l"   => "←"
 長い矢印
 "\>" => "⟶"
 二重矢印
 "\=>"  => "⇒"
 "\<=>" => "⇔"

 全称記号、存在記号
 "\all" => "∀"
 "\ex"  => "∃"

 等号
 Propositional equality
 "\==" => "≡"
 Setoid equality
 "\~~" => "≈"
 `Data.List.Relation.Binary.Equality.Setoid` の Equality
 "\~~~" => "≋"
 記号の下に等号を付けるときは
 記号 + "="
 "\sub=" => "⊆"
 "\?="   => "≟"

 否定
 "\neg" => "¬"
 二項関係の記号（等号など）を否定する場合
 二項関係の記号 + "n"

 関数合成
 "\o" => "∘"

 大小関係
 "\<=" => "≤"
 "\>=" => "≥"

 min max
 Levelの最大値
 "\lub" => "⊔"
 "\glb" => "⊓"

 ギリシャ文字
 "\G" + アルファベット
 "\Gl" => "λ"
 "\Ga" => "α"
 "\Gb" => "β"
 "\Gg" => "γ"
 "\GG" => "Γ"
 "\Go" => "ω"
 "\GS" => "Σ"
 "\Pi" => "Π"
 "\pi" => "π"
 "\phi" => "φ"

 下付き文字
 "\_" + 下につける文字
 "\_1" => "₁"
 "\_2" => "₂"

 上付き文字
 "\^" + 上につける文字
 注：エディタの下に候補が出てくるので選択すること
 "\^r" => "ʳ"
 "\^l" => "ˡ"

 乗算
 "\x" => "×"

 Divisibility
 絶対値はこの記号を使用している
 "\|" => "∣"

 論理
 "\or"  => "∨"
 "\and" => "∧"
 "\top" => "⊤"
 "\bot" => "⊥"

 集合
 "\in"  => "∈"
 "\sub" => "⊂"
 "\un"  => "∪"
 "\i"   => "∩"

 Equational Reasoning
 begin x ≡⟨ x≡y ⟩ y ∎
 "\<"   => "⟨"
 "\>"   => "⟩"
 "\qed" => "∎"

 括弧
 "\[[" => "⟦"
 "\]]" => "⟧"
 "\<<" => "⟪"
 "\>>" => "⟫"

 プライム　ダッシュ
 "\'" => "′"

 リストのコンス
 "\::" => "∷"

 チルダ
 "\~" => "∼"

 アルファベット修飾一覧
 "\B0"  => "𝟎"
 "\bA"  => "𝔸"
 "\MiA" => "𝐴"
 "\MIA" => "𝑨"
 "\McA" => "𝒜"
 "\MCA" => "𝓐"
 "\MfA" => "𝔄"
